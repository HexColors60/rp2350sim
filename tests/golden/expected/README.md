# Expected Output Files

Expected output files for comparison testing.

## File Types

- `.txt` - Text output (e.g., UART output)
- `.hex` - Hex dump of memory
- `.json` - Structured output
- `.bin` - Binary output

## Example

`uart_hello_expected.txt`:
```
Hello, RP2350!
GPIO blink test starting...
LED on
LED off
LED on
LED off
```
