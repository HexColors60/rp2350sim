# Icons Directory

This directory contains icon files used by the RP2350 Simulator.

## Icon Format

Icons should be PNG or SVG format, preferably at multiple resolutions:
- 16x16 for toolbar icons
- 24x24 for panel icons
- 32x32 for larger icons

## Required Icons

- `play.png` - Run simulation
- `pause.png` - Pause simulation
- `step.png` - Single step
- `reset.png` - Reset simulation
- `breakpoint.png` - Toggle breakpoint
- `memory.png` - Memory view
- `gpio.png` - GPIO panel
- `uart.png` - UART terminal
- `pio.png` - PIO inspector
- `trace.png` - Trace view
- `settings.png` - Settings
- `chip.png` - CPU view
