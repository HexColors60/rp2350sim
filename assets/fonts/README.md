# Fonts Directory

This directory contains font files used by the RP2350 Simulator.

## Required Fonts

- `monospace.ttf` - Monospace font for terminal and hex views
- `ui.ttf` - UI font for labels and buttons

## Recommended Fonts

- JetBrains Mono or Fira Code for monospace
- Inter or Roboto for UI

## Adding Fonts

Place the font files in this directory. The simulator will load them at startup.
